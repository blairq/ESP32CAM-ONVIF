## 🛠️ Build Process (PlatformIO + ESP-IDF)

**CLI Build via Makefile (Recommended):**
The project provides a Makefile that automates dependency management and compilation via a local virtual environment.

1. **Configure the Project:**
   ```bash
   make menuconfig
   ```

2. **Build the Firmware:**
   ```bash
   make build
   ```

3. **Flash the Firmware:**
   Ensure your device is connected via USB and run:
   ```bash
   make flash
   ```

## 📁 New Project Structure (ESP-IDF / PlatformIO)

```
ESP32CAM-ONVIF/
|-- boards/                   # Custom PlatformIO JSON board definitions
|-- components/               # Custom ESP-IDF components
|-- include/                  # C/C++ Header files (.h)
|   |-- config.h              # Auto-generated by gen_config.py
|-- src/                      # C/C++ Source files (.cpp)
|   |-- main.cpp              # ESP-IDF app_main entry point
|-- data/                     # SPIFFS Data
|-- platformio.ini            # PlatformIO configuration (espidf framework)
|-- CMakeLists.txt            # ESP-IDF Build system root
|-- Makefile                  # Automation CLI
```
